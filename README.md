# Посмотри в окно

Проектная работа по «Посмотри в окно».

## Ссылка на репозиторий

https://github.com/Osesiia/posmotri-v-okno-fd

## Описание

В проекте стилизован сайт с поиском видео по городу и времени суток. Реализована вёрстка основного экрана, формы поиска, списка карточек, состояний интерактивных элементов, прелоадера и блока ошибки.
